# Horror Cooked 9

An addon mod specifically for Sheltered 9 — 为「庇护所 9」量身定制的烹饪与玩法扩展模组。

| 属性 | 值 |
| --- | --- |
| Minecraft | 1.20.1 |
| Forge | 47.x (`forge_version=47.4.10`) |
| mod_id | `horrorcooked9` |
| Java 包 | `net.qiuyu.horrorcooked9` |
| 许可证 | All Rights Reserved |

## 快速开始

```bash
# 生成 IntelliJ 运行配置
./gradlew genIntellijRuns

# 生成 Eclipse 运行配置
./gradlew genEclipseRuns

# 构建
./gradlew clean build
```

产物位于 `build/libs/`。
